# Commands Used

## Verify Git Installation

```powershell
git --version
```

Purpose

Verifies whether Git is installed.

---

## Verify VS Code

```powershell
code --version
```

Purpose

Checks if Visual Studio Code is installed and available in PATH.

---

## Initialize Repository

```powershell
git init
```

Purpose

Creates a local Git repository.

---

## Check Repository Status

```powershell
git status
```

Purpose

Displays the current repository status.

---

## Stage Files

```powershell
git add .
```

Purpose

Stages all modified and new files.

---

## Create Commit

```powershell
git commit -m "Initialize DevOps portfolio structure"
```

Purpose

Creates a permanent snapshot.

---

## Add Remote Repository

```powershell
git remote add origin https://github.com/yuvraj1120/devops-portfolio.git
```

Purpose

Connects the local repository with GitHub.

---

## Verify Remote

```powershell
git remote -v
```

Purpose

Displays configured remote repositories.

---

## Push Repository

```powershell
git push -u origin main
```

Purpose

Uploads local commits to GitHub.
